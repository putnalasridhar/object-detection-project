# Real-Time Object Detection System

## Project Overview
This project implements a Real-Time Object Detection System using deep learning and computer vision. The system detects objects from a webcam video stream and displays the detected objects with bounding boxes and labels in real time.

The project uses the YOLOv8 model for fast and accurate object detection.

## Features
- Real-time object detection using webcam
- High accuracy detection using YOLOv8
- Bounding boxes with object labels
- Simple and easy interface
- Fast detection performance

## Technologies Used
- Python
- OpenCV
- YOLOv8
- NumPy

## Installation

Step 1: Clone the repository

git clone https://github.com/yourusername/object-detection-project.git

Step 2: Open the project folder

cd object-detection-project

Step 3: Install required libraries

pip install -r requirements.txt

## How to Run the Project

Run the following command:

python object_detection.py

After running the program, the webcam will open and start detecting objects in real time.

## Model Used
This project uses the YOLOv8 (You Only Look Once) deep learning model for object detection. The model can detect multiple objects such as person, mobile phone, bottle, chair, laptop and many more.
